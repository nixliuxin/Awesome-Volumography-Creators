## How to Add a Creator

### 1. Fork & Clone

```bash
git clone https://github.com/YOUR_USERNAME/Awesome-Volumography-Creators.git
cd Awesome-Volumography-Creators
npm install
```

### 2. Create Your Assets Folder

Create a folder named with your **creator ID** (lowercase, hyphenated):


**Naming Convention:**
- **profile.\*** â†’ Fixed name for avatar
- **cover.\*** â†’ Fixed name for cover image
- **gallery/\*** â†’ Any filename, **automatically detected!**

**Supported Formats:** `jpg`, `jpeg`, `png`, `webp`, `gif`

**Image Guidelines:**
- Avatar: ~400x400px, square
- Cover: ~1200x800px, 3:2 ratio
- Gallery: Reasonable file size (< 500KB each)

### 3. Add Entry to creators.json

Edit `website/data/creators.json`:

```json
{
  "id": "your-name",
  "name": "Your Name",
  "title": "Gaussian Splatting",
  "subtitle": "Optional secondary title",
  "location": "City / Country",
  "links": {
    "ct": "https://www.creative.tech/...",
    "website": "https://your-website.com",
    "twitter": "https://twitter.com/...",
    "instagram": "https://instagram.com/..."
  },
  "tags": ["4DGS", "VFX", "Spatial Computing"]
}
```

**That's it!** No need to list image paths â€” they're auto-detected by ID.

> ðŸ’¡ **Note:** The assets manifest is auto-generated by GitHub Actions when you submit a PR. No manual steps needed!

### 4. Test Locally (Optional)

```bash
npm run dev
```

### 5. Submit Pull Request

```bash
git add .
git commit -m "feat: add [Your Name] to creators"
git push origin main
```

Then open a Pull Request on GitHub.

## Opt-Out Policy

This is a **celebratory curation** of artists and researchers pushing the boundaries of volumetric media. We deeply respect everyone's preferences regarding their online presence.

**If you prefer not to be listed:**

1. **GitHub Issue** (Recommended)  
   [Open a removal request](https://github.com/nixliuxin/Awesome-Volumography-Creators/issues/new?title=Removal%20Request:%20[Your%20Name]&labels=removal) with the title "Removal Request: [Your Name]"

2. **Direct Contact**  
   Reach out via [Creative.Tech](https://creative.tech)

**We will remove your entry within 48 hours, no questions asked.**

> ðŸ’¡ This list is non-commercial and purely for community knowledge-sharing. We add creators based on their publicly visible work in the volumetric/3D scanning space.
